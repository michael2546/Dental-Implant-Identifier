Why Support

For more accurate prediction, the deep learning model should be trained on as large a dataset as possible. This allows the model to distinguish the features and characteristics of each implant more effectively.

Our goal is to increase the number of supported implant types that our model can predict. To achieve this, we need to collect new labeled implant image data.

How to Support

To support our project, please follow these steps:

  1. Select an implant image from your device
  2. Enter the implant type in the field provided
  3. Click the submit button
